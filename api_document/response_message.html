    建檔/比對回傳格式

    ! 需要在您的外部程式中也架設用於接收任務處理結果的Named Pipe Server。

    不同於 Api 頁面執行的結果會立即回傳，此處會等到指派的任務(建檔/比對)執行完成或者中斷後，才會將最後結果回傳， 在此將各個任務回傳的結果格式與說明。

    以下所有可能的回傳結果：

    • 虹膜建檔回傳結果
    • 虹膜比對回傳結果

    虹膜建檔回傳結果

    執行虹膜建檔完成，或者是中斷建檔時的回傳訊息

    Returns:

    Name Type Description
    Status str

    執行狀態，success or error
    StatusCode int

    執行狀態碼，0 or 1，0為成功，1為失敗
    Message str

    執行訊息
    Action str

    執行動作
    Data dict

    執行結果
    season_id str

    賽季編號
    loft_id str

    舍號
    feet_number str

    腳環編號
    folder_path str

    建檔資料夾的最上層位置
    build_num int

    建檔張數
    complete_num int

    完成建檔張數
    save_path str

    儲存路徑
    save_file_name_list array

    建檔檔案名稱列表
    Response Sample
        {
        "Status": "success",
        "StatusCode": 0,
        "Message": "建檔成功!",
        "Action": "build_iris",
        "Data": {
            "season_id": "2021",
            "loft_id": "A",
            "feet_number": "001",
            "folder_path": "D:\iris",
            "build_num": 3,
            "complete_num": 1,
            "save_path": "D:\iris\2021\A\001\",
            "save_file_name_list": [
                "R-rand_uid_1.jpg",
                "R-rand_uid_2.jpg",
                "R-rand_uid_3.jpg"
            ]
        }
    }

    虹膜比對回傳結果

    執行虹膜比對完成，或者是中斷比對時的回傳訊息

    Returns:

    Name Type Description
    Status str

    執行狀態，success or error
    StatusCode int

    執行狀態碼，0 or 1，0為成功，1為失敗
    Message str

    執行訊息
    Action str

    執行動作
    Data dict

    執行結果
    season_id str

    賽季編號
    loft_id str

    舍號
    feet_number str

    腳環編號
    folder_path str

    建檔資料夾的最上層位置
    threshold float

    比對閥值
    compare_times int

    指定可比對次數
    need_save bool

    是否需要儲存比對結果
    compare_pass bool

    比對是否通過
    current_compare_times float

    本輪比對花費的次數
    min_compare_value float

    本輪比對所得到的最小數值
    Response Sample
        {
        "Status": "success",
        "StatusCode": 0,
        "Message": "比對成功!",
        "Action": "compare_iris",
        "Data": {
            "season_id": "2021",
            "loft_id": "A",
            "feet_number": "001",
            "folder_path": "D:\iris",
            "threshold": 0.3,
            "compare_times": 3,
            "need_save": true,
            "compare_pass": true,
            "current_compare_times": 3,
            "min_compare_value": 0.2
        }
    }
