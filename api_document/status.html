    程式畫面說明

    以下針對各個畫面做其對應狀態的詳細的說明

    載入

    此畫面代表程式在進行載入 相關的內容有 判斷網路是否連接、 載入辨識模型、 GPU 裝置調用和測試

    !注意，在程式載入完成前所有API 指令皆無法生效
    載入完成後會主動發訊息到您的Named Pipe Server，詳細回傳格式可參考 載入任務回傳結果

    歡迎使用

    程式載入完成並可開始使用的畫面

    網路連線失敗

    如果程式判斷當前網路環境無法連線
    則會顯示此畫面

    攝影機載入中

    當程式在嘗試與攝影機晶片建立連接
    或是等待另外一個攝影機連上線時
    會顯示以下畫面

    攝影機連線失敗

    當檢測到攝影機連線失敗
    比如: 線材脫落、晶片損毀
    會顯示以下畫面

    該攝影機禁止使用

    當建檔的left_build_num / right_build_num 其一設定為0
    或是比對時，有透過set_compare_focus_camera 指定要focus的鏡頭
    被禁止使用的攝影機會顯示以下畫面
