    .env.secrect 環境設定檔說明

    介接程式底下setting/ 資料夾中有個.env.secrect檔案，用途和.env相同都是軟體環境參數
    差別在於，.env.secrect的設定更不希望一般使用者去做更改
    可能會造成使用上的錯誤

    BUILD_IRIS_BLUR_THRESHOLD

    此參數提供給建檔時使用的模糊度閾值，數值越高需要越清晰的影像
    預設 '450'

    COMPARE_IRIS_BLUR_THRESHOLD

    此參數提供給辨識時使用的模糊度閾值，數值越高需要越清晰的影像
    預設 '450'

    BUILD_CLEAR_PHOTO_LIMIT

    此參數設定了當建檔時需要等待幾幀的清晰影像後才開始採樣
    預設 '15'

    COMPARE_CLEAR_PHOTO_LIMIT

    此參數設定了當比對時需要等待幾幀的清晰影像後才開始比對
    預設 '5'

    DETECT_FRAME_COOL_DOWN

    此參數設定了每次框選完眼睛後，要休息幾幀才進行下次眼睛框選
    提供給較低階顯卡使用，比如使用時發現幀數很低，可適當加入cool_down
    讓框選眼睛的頻率降低。
    預設 '0'

    USE_NEW_DETECT_METHOD

    當前系統中有兩套虹膜分析方法，新方法對於半遮以及無法進行建檔比對的分析結果做出阻擋。
    但仍需測試，如有無法分析的情況產生可藉由調整此參數，讓系統預設使用舊方法分析。
    1: 使用 0: 不使用
    預設 '1'

    USE_EMA_UPDATE_METHOD

    此參數設定了是否使用EMA的方法來更新檢測過濾如"眨眼"/"晃動"的閾值，
    開啟後可更適配於各種不同的光源以及鴿種，
    但較為適用於鴿子進入鏡頭後能正常睜眼
    若是對於不斷眨眼或是眼睛睜不開的拍攝情境來說效果較差
    1: 使用 0: 不使用
    預設 '1'
