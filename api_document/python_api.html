    基本視窗設定

    設定視窗位置

    跟據傳入的x,y座標設定視窗位置

    Attributes:

    Name Type Description
    Action str

    執行動作，setPosition
    Data dict

    傳入的資料
    x int

    視窗x座標
    y int

    視窗y座標
    Request Sample
        {
        "Action": "setPosition",
        "Data": {
            "x": 100,
            "y": 100
        }
    }

    Returns:

    Name Type Description
    Status str

    執行狀態，success or error
    StatusCode int

    執行狀態碼，0 or 1，0為成功，1為失敗
    Message str

    執行訊息
    Response Sample
        {
        "Status": "success",
        "StatusCode": 0,
        "Message": "Action 'setPosition' executed successfully"
    }

    虹膜建檔設定

    開始虹膜建檔

    此處只負責指派虹膜建檔任務到執行緒中，如果需要查看建檔回傳結果請查看 虹膜建檔回傳結果

    開始執行虹膜建檔

    Attributes:

    Name Type Description
    Action str

    執行動作，startBuildIris
    Data dict

    傳入的資料
    season_id str

    賽季編號
    loft_id str

    舍號
    feet_number str

    腳環編號
    folder_path str

    建檔資料夾的最上層位置
    build_num int

    (Optional) 建檔張數，預設為3
    use_left_eye bool

    (Optional) 是否使用左眼，預設為True
    Request Sample
        {
        "Action": "startBuildIris",
        "Data": {
            "season_id": "2021",
            "loft_id": "A",
            "feet_number": "001",
            "folder_path": "D:\iris_data",
            "build_num": 3,
            "use_left_eye": True
        }
    }

    Returns:

    Name Type Description
    Status str

    執行狀態，success or error
    StatusCode int

    執行狀態碼，0 or 1，0為成功，1為失敗
    Message str

    執行訊息
    Response Sample
        {
        "Status": "success",
        "StatusCode": 0,
        "Message": "Action 'startBuildIris' executed successfully"
    }
