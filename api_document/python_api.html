    基本視窗設定

    設定視窗位置

    根據傳入的x,y座標設定視窗位置

    Attributes:

    Name Type Description
    Action str

    執行動作，setPosition
    Data dict

    傳入的資料
    x int

    視窗x座標
    y int

    視窗y座標
    Request Sample
        {
        "Action": "setPosition",
        "Data": {
            "x": 100,
            "y": 100
        }
    }

    Returns:

    Name Type Description
    Status str

    執行狀態，success or error
    StatusCode int

    執行狀態碼，0 or 1，0為成功，1為失敗
    Action str

    執行動作
    Message str

    執行訊息
    Response Sample
        {
        "Status": "success",
        "StatusCode": 0,
        "Action": "setPosition",
        "Message": "Action 'setPosition' executed successfully"
    }

    開啟攝影機

    開啟攝影機 single_camera_mode: 要採用的單鏡頭模式
    "left" "right" 或 "disable" 預設為"disable"

    disable: 以雙鏡頭模式開啟
    left: 以左側鏡頭開啟
    right: 以右側鏡頭開啟

    Attributes:

    Name Type Description
    Action str

    執行動作，openCamera
    Data dict

    傳入的資料
    single_camera_mode str

    (Optional) 單鏡頭模式，"left" "right" 或 "disable" 預設為"disable"
    Request Sample
        {
        "Action": "openCamera",
        "Data": {
            single_camera_mode: "disable"
        }
    }

    Returns:

    Name Type Description
    Status str

    執行狀態，success or error
    StatusCode int

    執行狀態碼，0 or 1，0為成功，1為失敗
    Action str

    執行動作
    Message str

    執行訊息
    Response Sample
        {
        "Status": "success",
        "StatusCode": 0,
        "Action": "openCamera",
        "Message": "Action 'openCamera' executed successfully"

    開始執行虹膜建檔

    Attributes:

    Name Type Description
    Action str

    執行動作，startBuildIris
    Data dict

    傳入的資料
    season_id str

    賽季編號
    loft_id str

    舍號
    feet_number str

    腳環編號
    folder_path str

    建檔資料夾的最上層位置
    left_build_num int

    (Optional) 左鏡頭建檔張數，預設為3，最小為0
    right_build_num int

    (Optional) 右鏡頭建檔張數，預設為3，最小為0

    虹膜比對設定

    開始虹膜比對

    此處只負責指派虹膜比對任務到執行緒中，如果需要查看比對回傳結果請查看 虹膜比對回傳結果 注意!! 如果鏡頭開啟失敗或正在啟動，會無法執行此接口

    開始執行虹膜比對

    Attributes:

    Name Type Description
    Action str

    執行動作，startCompareIris
    Data dict

    傳入的資料
    season_id str

    賽季編號
    loft_id str

    舍號
    feet_number str

    腳環編號
    folder_path str

    建檔資料夾的最上層位置
    threshold float

    (Optional) 比對門檻值，介於0.30 ~ 0.50 越高越寬鬆 預設為0.35
    compare_times int

    (Optional) 比對次數 預設為10
    need_save bool

    (Optional) 如果比對相符，是否儲存本次比對的分析檔，擴增資料庫 預設為False
    Request Sample
        {
        "Action": "startCompareIris",
        "Data": {
            "season_id": "2021",
            "loft_id": "A",
            "feet_number": "001",
            "folder_path": "D:\iris_data",
            "threshold": 0.35,
            "compare_times": 10
            "need_save": False
        }
    }

    Returns:

    Name Type Description
    Status str

    執行狀態，success or error
    StatusCode int

    執行狀態碼，0 or 1，0為成功，1為失敗
    Action str

    執行動作
    Message str

    專注於特定鏡頭

    讓比對時能專注於特定鏡頭，設定後會自動關閉另一鏡頭，並只以該鏡頭進行處理，
    比如設定為left，那程式就只會抓取位於畫面左側的鏡頭做比對

    Attributes:

    Name Type Description
    Action str

    執行動作，setCompareFocusCamera
    Data dict

    傳入的資料
    camera_src str

    要聚焦的鏡頭，"left" 或 "right"
    Request Sample
        {
        "Action": "setCompareFocusCamera",
        "Data": {
            "camera_src": "right"
        }
    }

    Returns:

    Name Type Description
    Status str

    執行狀態，success or error
    StatusCode int

    執行狀態碼，0 or 1，0為成功，1為失敗
    Action str

    執行動作
    Message str

    執行訊息
    Response Sample
        {
        "Status": "success",
        "StatusCode": 0,
        "Action": "setCompareFocusCamera",
        "Message": "Action 'setCompareFocusCamera' executed successfully"
